# DinaMenuDesigner
Éditeur visuel de menus pour le framework MonoGame DinaCSharp.

## Description
DinaMenuDesigner est une application WPF qui permet de concevoir visuellement 
les menus du framework DinaCSharp. Elle offre un aperçu en temps réel sur un 
canvas 1920x1080, avec la possibilité de positionner les éléments à la souris.

## Fonctionnalités
- Création et gestion de titres et d'items de menu
- Aperçu en temps réel sur un canvas 1920x1080
- Drag & Drop pour repositionner les éléments
- Mode groupé pour les items (bloc déplaçable)
- Édition des propriétés (texte, position, dimensions, couleur, police, ombre, etc.)
- Contrainte de taille minimale basée sur la taille du texte
- Sélecteur de couleur RGBA intégré
- Support des alignements horizontaux et verticaux
- Sauvegarde et chargement de projets en JSON
- Génération de code C#

## Technologies
- .NET 10
- WPF (Windows Presentation Foundation)
- Architecture MVVM
- System.Text.Json

## En développement

- Gestion des polices SpriteFonts MonoGame
- Menu applicatif
