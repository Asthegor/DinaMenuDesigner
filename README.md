# DinaMenuDesigner

Éditeur visuel de menus pour le framework MonoGame DinaCSharp.

## Description

DinaMenuDesigner est une application WPF qui permet de concevoir visuellement les menus du framework DinaCSharp. Elle offre un aperçu en temps réel sur un canvas 1920x1080, avec la possibilité de positionner les éléments à la souris.

## Fonctionnalités

- Création et gestion de titres et d'items de menu
- Aperçu en temps réel sur un canvas 1920x1080
- Drag & Drop pour repositionner les éléments
- Édition des propriétés (texte, position, couleur, police, ombre, etc.)
- Sélecteur de couleur RGBA intégré
- Support des alignements horizontaux et verticaux
- Sauvegarde et chargement de projets en JSON
- Génération de code C#

## Technologies

- .NET 10
- WPF (Windows Presentation Foundation)
- Architecture MVVM
- System.Text.Json (à venir)

## En développement
